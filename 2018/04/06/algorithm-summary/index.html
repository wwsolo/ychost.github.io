位运算

  1. 判断是否为 2 的幂 
    (num & (num-1)) == 0
  2. 判断一个数是否为偶数 
    (num & 1) == 0
  3. 将一个数乘除 2 
    // 乘2
num << 1
// 除2，注意是否需要最高位，否则用 >>>
num >> 1
  4. 将一个数的二进制状态下最右边的 1 变成 0 
    num = num & (num - 1)
  5. 将一个字符改变大小写 
    ch = (char)(ch ^ 32)
